Pilih metode yang sesuai dengan perangkatmu

Panduan untuk pengguna Android/Emulator

  1. Unduh Virtual Host
  2. Unduh File Host GTPS
  3. Pasang Virtual Host di perangkatmu
  4. Buka Virtual Host dan klik "PILIH FILE HOST"
  5. Pilih Gtps File Host.txt
  6. Nyalakan Virtual Host
  7. Buka Growtopia!!

Panduan untuk pengguna Android/Emulator

  1. Unduh Power Tunnel
  2. Install Power Tunnel ke perangkat Anda
  3. Buka Power Tunnel
  4. Buka aplikasi Powertunnel ‚Üí ikon tiga garis ‚Üí Host Settings.
  5. Tempel URL host yang sudah disalin.
  6. Ubah host update period jadi on start.
  7. Aktifkan Powertunnel, kemudian login ke GTPS kamu.

Panduan untuk pengguna Android/Emulator

  1. Unduh Host Go
  2. Install Hosts Go ke perangkat Anda.
  3. Buka Host GO dan Pilih 'Hosts Editor'.
  4. Klik ‚ãÆ di sebelah kanan, dan klik 'Download Hosts File'
  5. Pilih 'Download and Apply'
  6. Kembali dan klik 'START'
  7. Buka Growtopia!!

Panduan untuk pengguna iPhone/iPad

  1. Buka App Store dan unduh Surge 5
  2. Buka Surge 5 dan Klik 'OKE'
  3. Klik 'Default.conf'
  4. Pilih 'IMPORT' > 'Download Profile From URL'
  5. Tekan 'OK' dan 'DONE'
  6. Klik 'SETUP' setujui kebijakan klik 'OK'
  7. Tekan 'Allow' untuk konfigurasi VPN
  8. Buka Growtopia!!

Panduan untuk pengguna MacBook

  1. Buka Finder lalu di menu pilih Go ‚Üí Go to Folder.
  2. Ketik path berikut: /private/etc/hosts
  3. Salin file 'hosts' ke Desktop agar bisa diedit.
  4. Buka file 'hosts' di Desktop menggunakan TextEdit atau editor teks lain.
  5. Tambahkan baris berikut di bagian bawah file:
    6. 175.246.23 www.growtopia1.com
    175.246.23 www.growtopia2.com
  6. Simpan file lalu pindahkan kembali ke folder /private/etc/ (replace file lama).
  7. Jika muncul permintaan izin, klik 'Authenticate' dan masukkan password Mac.
  8. Opsional: buka Terminal dan jalankan perintah berikut untuk mempercepat pembacaan:sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
  9. Buka Growtopia dan pastikan sudah terhubung ke GTPS!

Panduan untuk pengguna Windows PC/Laptop

  1. Tekan ‚äû Win+R (Windows Key + R)
  2. Ketik C:\Windows\System32\drivers\etc
  3. Temukan file yang bernama hosts
  4. Klik kanan file hosts dan klik 'Edit with Notepad++'
  5. Copy dan paste IP berikut:
    6. 175.246.23 www.growtopia1.com
    175.246.23 www.growtopia2.com
  6. Tekan Ctrl + S untuk menyimpan file
  7. Buka Growtopia!!
